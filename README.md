# API Automation Testing with REST Assured (Advanced)

## 📌 Deskripsi Project
Project ini adalah latihan lanjutan (Exercise Day 34) untuk memperdalam keterampilan *automation testing* REST API. Fokus utama dari project ini adalah membangun *automation test* yang *robust*, *scalable*, dan siap diintegrasikan ke dalam *workflow* DevOps.

## 🎯 Objektif
* **Authentication & Authorization:** Mengimplementasikan pengujian API dengan mekanisme keamanan (Basic Auth / Bearer Token).
* **Handling Dynamic Data:** Mengekstraksi dan menangani data yang berubah secara dinamis (seperti menyimpan token dari *response* *login* untuk *request* selanjutnya).
* **Data-Driven Testing:** Menggunakan eksternal *file* (JSON/CSV) untuk menguji API dengan berbagai variasi *input* secara otomatis.
* **CI/CD Integration:** Mempersiapkan *automation test* agar dapat berjalan secara otomatis di *pipeline* CI/CD (seperti Jenkins/GitLab CI) setiap kali ada *commit* baru.

## 🛠️ Tools & Technologies
* **Language:** Java
* **Build Tool:** Maven
* **Testing Framework:** TestNG
* **API Framework:** REST Assured
* **IDE:** IntelliJ IDEA
* **API Client:** Postman (untuk eksplorasi manual)
